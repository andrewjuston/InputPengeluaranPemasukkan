💰 Expense Tracker (VB .NET)

A simple desktop-based Expense Tracker application built using VB .NET.
This application allows users to manage daily expenses with basic CRUD operations and Excel file integration.

📌 Features

✅ Create new expense records

✅ Read / View all expense records

✅ Update existing expense entries

✅ Delete expense entries

✅ Import expense data from Excel (.xlsx)

✅ Export expense data to Excel (.xlsx)

✅ Simple and user-friendly interface

🛠️ Built With

VB .NET

Windows Forms

Microsoft Excel Interop / EPPlus (depending on your implementation)
